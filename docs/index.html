  Install

  1. Install NodeJS 6.x

    Follow instruction on NodeJS official page.
    Installing Node.js (nodejs.org)  

  2. Install sssg globally (You can also install locally)
    $ sudo npm install -g git+https://github.com/Hinaser/sssg

  Initialize

  1. Create project directory
    $ mkdir sample
    $ cd ./sample
  2. Initialize source directory
    $ sssg init
    $ ls -l

    Confirm that ./src directory has been created

  Develop

Now you are ready to develop! Type "sssg serve" like the following.

$ sssg serve

Browser will launch a web page for you.

As soon as you edit source files in "./src" folder, the latest change will be live-reloaded on your browser's page.
